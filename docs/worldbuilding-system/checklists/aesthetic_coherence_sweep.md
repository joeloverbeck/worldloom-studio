# Aesthetic Coherence Sweep

Serves `17_aesthetic_coherence_and_semiosis.md`. Use when a fact, region, institution, branch, protected effect, or adaptation changes tone, symbols, language, sensory texture, recurrence, wonder, sacred opacity, horror, symbolic excess, or genre contract.

## Causal texture

- [ ] Fact, region, institution, branch, protected effect, or scene family named.
- [ ] Mood and genre pressure stated.
- [ ] Consequence mode named where relevant.
- [ ] The aesthetic promise is tied to world causes, not decoration.
- [ ] Recurring motifs identified.
- [ ] Symbolic transformations identified.
- [ ] Official symbols and counter-symbols identified.
- [ ] Wonder, sacred opacity, horror/dread, or symbolic excess named if present.

## Channels

- [ ] Names affected: places, personal names, titles, insults, euphemisms, eras, festivals.
- [ ] Materials affected: cloth, metal, wood, stone, bone, glass, paper, food, tools, waste.
- [ ] Architecture or space affected: thresholds, roads, rooms, borders, sacred zones, ruins, forbidden maps.
- [ ] Costume/body presentation affected: clothing, scars, masks, tattoos, hair, scent, posture, prosthetics.
- [ ] Language or speech practice affected: registers, taboos, accents, formulae, jargon, proverbs, jokes, curse words, silences.
- [ ] Ritual or performance affected: gestures, ceremonies, games, songs, alarms, court actions, daily routines, rites of refusal.
- [ ] Soundscape, color, light, weather, animal sign, transformation, or silence affected where relevant.

## Variation and residue

- [ ] Regional, class, institutional, species, branch, or era variation explained.
- [ ] Aesthetic fossils listed for longstanding facts.
- [ ] Propaganda and counter-symbols show who fights over meaning.
- [ ] Decorative details without cause removed or given cause.
- [ ] Non-instrumental wonder is protected from being converted into a puzzle unless a puzzle is intended.
- [ ] Horror/dread is not domesticated by premature explanation.
- [ ] Symbolic excess is consistent without being reduced to one gloss.
- [ ] Aesthetic drift is either repaired or deliberately governed.

## Version 0.5 completion rule

Parent protocol: `17_aesthetic_coherence_and_semiosis.md`.

A completed sweep must leave `a named aesthetic pressure, consequence mode where relevant, at least one causal residue, any preservation boundary, and a drift decision`. If the sweep is skipped for a major fact, record why the parent protocol did not apply.

## Red flags

- The checklist is used as a yes/no ritual without recording consequences.
- A box is checked because the author likes the image rather than because the world now answers it.
- A symbol is only decoration.
- A protected wonder becomes a clue hunt by accident.
- Sacred opacity is treated as lack of design rather than a governed boundary.
- Horror is explained before dread has consequences.
- The sweep discovers canon debt but no follow-up is named.
- The sweep changes canon status, constraint tags, or repair operations without using the proper parent protocol.
