# Agent / Character Sweep

Serves `11_agent_character_psychology.md`. Use when a world fact changes actor behavior, identity, belief, opportunity, cost, fear, or story pressure.

## Actor pressure

- [ ] Affected actors named: beneficiary, loser, gatekeeper, excluded person, enforcer, believer, skeptic, fraud, victim, child, elder, criminal, reformer, collaborator, rebel.
- [ ] Desires identified.
- [ ] Fears identified.
- [ ] Identity commitments identified.
- [ ] Knowledge, ignorance, misbelief, and refusal to know separated.
- [ ] Capabilities listed.
- [ ] Constraints listed: body, law, money, trauma, time, language, geography, reputation, oath, relationship, taboo.
- [ ] Incentives listed.
- [ ] Relationships considered.

## Bounded agency

- [ ] Awareness gate checked.
- [ ] Interpretation gate checked.
- [ ] Capacity gate checked.
- [ ] Incentive gate checked.
- [ ] Permission gate checked.
- [ ] Bounded rationality considered: what can the actor actually know and process?
- [ ] Social identity considered: what group belonging shapes action?
- [ ] Norms and perceived control considered.
- [ ] Hostile optimization and situated agency both considered.

## Character consequence

- [ ] Character behavior tested against world pressure.
- [ ] No actor changes only because the plot needs it.
- [ ] Contradictory behavior repaired, assigned to truth layers, or made deliberately motivated.
- [ ] Psychological residue recorded: habits, trauma, pride, shame, denial, superstition, gallows humor, identity politics.
