# Constraint Composition Sweep

Use when multiple limits govern the same fact.

- [ ] Main constrained fact named.
- [ ] Each constraint stated in plain prose.
- [ ] Constraint type identified.
- [ ] Constraints tested for stacking: do they strengthen each other?
- [ ] Constraints tested for cancellation: does one make another irrelevant?
- [ ] Constraints tested for contradiction.
- [ ] Bottleneck identified.
- [ ] Loopholes listed.
- [ ] Enforcement named.
- [ ] Bypass actors named.
- [ ] Ordinary-life residue of the limits identified.
- [ ] The constrained fact still remains worth keeping.
